# Variables y Constantes en Python

## ¿Qué es una variable?

Una **variable** es un espacio en memoria que almacena un valor.  
Podés pensarla como una caja con un nombre que guarda información.

Ejemplo:

```python
nombre = "Ian"
edad = 18
```

## Reglas para nombrar variables

Deben empezar con una letra o guion bajo (_), nunca con un número.

Solo pueden contener letras, números y guiones bajos.

Python distingue entre mayúsculas y minúsculas (edad ≠ Edad).

No deben usar palabras reservadas del lenguaje (como if, while, class, etc.).

Usar nombres descriptivos es buena práctica.

Ejemplos válidos:

```python
nombre_usuario = "Carlos"
edad = 20
_total = 100
```

Ejemplos inválidos:

```python
2numero = 10     # comienza con número
nombre-usuario = "Juan"  # contiene un guion
if = 5           # palabra reservada
```

## Asignación de variables

Se usa el signo = para asignar un valor.

```python
x = 10
y = 20
suma = x + y
```

También se puede asignar el mismo valor a varias variables:

```python
a = b = c = 0
```

O múltiples valores a la vez:

```python
nombre, edad, pais = "Lucía", 22, "Argentina"
```

## Tipado dinámico

En Python, las variables pueden cambiar de tipo durante la ejecución.
El lenguaje detecta automáticamente el tipo según el valor asignado.

```python
x = 10       # int
x = "Hola"   # ahora es str
```
