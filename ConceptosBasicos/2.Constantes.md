# Constantes

Python no tiene constantes verdaderas (no hay una palabra reservada como const).
Por convención, se usan nombres en mayúsculas para indicar que una variable no debería modificarse.

```python
PI = 3.1416
GRAVEDAD = 9.8
```

Aunque se pueden cambiar, se considera una mala práctica hacerlo.

## Eliminación de variables

Podés eliminar una variable con la instrucción del:

```python
x = 100
del x
```

Intentar usarla después generará un error.
