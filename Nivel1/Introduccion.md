# Introducción a Python

## ¿Qué es Python?

Python es un lenguaje de programación de **alto nivel**, **interpretado** y **multipropósito**.  
Se caracteriza por una **sintaxis simple y legible**, lo que lo convierte en una excelente opción tanto para principiantes como para desarrolladores avanzados.

Python se utiliza en múltiples áreas:
- Desarrollo web
- Análisis de datos
- Inteligencia artificial y aprendizaje automático
- Automatización de tareas
- Desarrollo de software y videojuegos
- Ciberseguridad
- Ciencia y educación

---

## Historia de Python

- **1989**: Guido van Rossum comienza el desarrollo de Python en los Países Bajos.  
- **1991**: Se publica la primera versión (Python 0.9.0).  
- **2000**: Llega Python 2.0, que introduce muchas mejoras.  
- **2008**: Se lanza Python 3.0, una versión completamente renovada.  
- Hoy en día, Python 3 es el estándar y Python 2 ya no se mantiene.

El nombre “Python” no proviene del animal, sino del grupo de comedia británico **“Monty Python”**.

---

## Características principales

- **Sintaxis clara y sencilla**  
- **Tipado dinámico**: no es necesario declarar el tipo de las variables  
- **Multiplataforma**: funciona en Windows, macOS y Linux  
- **Gran comunidad y librerías** disponibles  
- **Interpretado**: no requiere compilación previa  
- **Orientado a objetos**, pero también admite programación funcional y procedural

---

## Instalación de Python

1. Ingresar a [https://www.python.org/downloads/](https://www.python.org/downloads/)  
2. Descargar la versión más reciente de Python 3  
3. Durante la instalación, marcar la opción **“Add Python to PATH”**  
4. Finalizar el proceso y comprobar la instalación

Para verificar que se instaló correctamente, abrir una terminal y escribir:

```bash
python --version
```

O en otros sistemas: 

```bash
python3 --version
```

## Ejecutar codigo

Desde la terminal:

```bash
python archivo.py
```
